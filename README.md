# Dependências

- ReactJS
- @fortawesome/fontawesome-svg-core
- @fortawesome/free-brands-svg-icons
- @fortawesome/free-solid-svg-icons
- @fortawesome/react-fontawesome

# Configurações

É necessário configurar as **variavéis de ambiente** no **.env**

```
REACT_APP_URL_BASE
REACT_APP_API_KEY
REACT_APP_API_BASE
REACT_APP_LANGUAGE
```
